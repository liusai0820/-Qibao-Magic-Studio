# 皮克斯3D模式使用指南

## 功能介绍

皮克斯3D模式是一个全新的创作风格，用于生成皮克斯电影风格的3D动画场景。特别适合生成人物自拍、合照等欢乐互动场景。

## 新增主题

- 🎬 **皮克斯3D** - 通用皮克斯3D自拍场景
- 📸 **三国自拍** - 三国英雄人物自拍
- 🎭 **历史人物** - 历史人物自拍
- 🌟 **明星合照** - 明星合照自拍

## 使用示例

### 示例 1：三国英雄自拍

**输入主题：** `三国英雄自拍`

**生成的场景会包含：**
- 刘备、张飞、关羽三位英雄
- 刘备站在中央手持自拍杆
- 张飞在左侧做出豪迈姿势
- 关羽在右侧展现威武气质
- 古风建筑或山林背景
- 皮克斯电影级的3D渲染效果

### 示例 2：自定义人物组合

**输入主题：** `孙中山、周恩来、毛泽东领导人自拍`

系统会自动生成包含这些人物的欢乐自拍场景。

## 提示词结构

皮克斯3D模式的提示词包含以下关键元素：

```
【人物组合】在【场景环境】中欢乐自拍留念

【主视角人物】：
- 站在中央，手持自拍杆
- 表情为【性格特征】
- 呈现出【领导/核心人物气质】

【左侧人物】：
- 展现出【活泼/友好】的动作/姿势
- 表情为【开心/兴奋】
- 体现出【配角性格】

【右侧人物】：
- 摆出【搞怪/豪迈/可爱】的姿势
- 手持【代表物品】
- 表情为【欢乐/调皮】
- 风格活泼夸张
```

## 人物名称修改

在生成后，系统会自动识别人物并在最后修改人物名称。你也可以手动指定：

**输入示例：**
```
刘备、张飞、关羽在古风建筑前自拍
```

**生成结果会包含：**
- 刘备（中央，领导气质）
- 张飞（左侧，豪迈气质）
- 关羽（右侧，威武气质）

## 场景环境选项

可以指定不同的背景环境：

- 🏯 **古风建筑** - 宫殿、城堡、古刹
- 🌸 **自然风景** - 桃花林、湖泊、山林
- 🌌 **仙境场景** - 云海、仙境、魔法森林
- 🏞️ **现代场景** - 城市、公园、度假地

## 技术参数

- **风格：** Pixar 3D Animation Style
- **渲染质量：** 8K Ultra HD
- **光效：** 柔和温暖的电影级光效
- **色彩：** 明亮鲜艳的皮克斯色调
- **构图：** 电影级构图感

## 常见用途

1. **历史人物合照** - 生成历史名人的欢乐自拍
2. **文学人物互动** - 生成文学作品中人物的自拍场景
3. **家庭成员合照** - 生成家庭成员的温馨自拍
4. **团队合照** - 生成团队成员的欢乐合照
5. **角色扮演** - 生成特定角色的自拍场景

## 提示

- 人物名称会在最后自动修改，确保准确性
- 场景环境会根据主题自动调整
- 所有人物表情和动作都会充满个性和活力
- 整体画面充满温馨、欢乐的气息
